Installing TabGen

To install TabGen, simply copy its files into the appropriate directory on your computer. The process is detailed here. Note that TabGen is an add-in, not a script. The files that need to be copied are the ones in the distribution zip file. Once its files have been copied to the right place, TabGen will be available from Fusion 360’s Modify menu the next time Fusion 360 is started.

See help.html for information on using TabGen.
